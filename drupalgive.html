Giving Back to Drupal

Drupal Give

At OpenConcept, we use Drupal in all of our work, but we also contribute back to Drupal by writing modules, themes, participating in forums, lecturing in conferences, organizing Drupal events and anything else we can think of.
This page displays the highlights of our give-backs to the community. It is our part of the /drupalgive initiative.

Our organization is not only using Drupal, but also contributes back to the community and to the project by writing modules and themes, participating in forums, lecturing in conferences, organizing Drupal events and more.

Clean Markup module contributions

Clean Markup is a module that aims to clean up, enhance and facilitate the customization of markup for Drupal core and several popular contrib modules. Besides making HTML markup cleaner and easier to theme, it can also be used to increase the accessibility of pages.

OpenConcept regularly pays Matt, a co-maintainer of the module, to contribute bug-fixes and add features to Clean Markup.

Ottawa Drupal in Government Showcase

Organized a Drupal in Government showcase featuring Agriculture Canada, the Department of Defense, the Ottawa Public Library and CIPPIC. This was an important networking event for public servants who needed examples of other departments using Drupal.

OpenLayers Module

Mike has contributed a number of patches and overhauled the documentation for the Drupal 7 implementation of OpenLayers. In looking at ways to improve accessibility in OpenLayers, Mike also contributed back to the main Javascript library and blogged about the process.

Web Experience Toolkit (WET) Drupal Variant

Mike has been contributing to the Web Experience Toolkit of the Government of Canada.  Before StatsCan had taken leadership over the Drupal Distribution, OpenConcept had contributed a Zen based CLF 2.0 theme, but we're very happy to see the direction that the project has gone since it expanded to include workflow & Apps.
